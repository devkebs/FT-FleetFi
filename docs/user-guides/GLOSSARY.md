# FleetFi Glossary

| Term | Definition |
|------|------------|
| Asset | Physical fleet component (EV, swap station) that produces telemetry & revenue. |
| Token | Fractional ownership unit representing a share of an asset, minted on-chain. |
| Minting | Creation of a new token with blockchain record (tx_hash, chain). |
| Wallet | Internal ledger for user balances and transactions (NGN). |
| Payout | Distribution of accumulated revenue to token holders' wallets. |
| Telemetry | Real-time operational data (location, SOH, ride stats, swaps). |
| SOH | State of Health; battery/vehicle condition metric affecting performance. |
| Swap Event | Battery exchange operation tracked for stations/vehicles. |
| KYC | Know Your Customer identity verification process (IdentityPass provider). |
| RBAC | Role-Based Access Control managing permissions (investor/operator/admin). |
| Notification | System-generated event message (investment, payout, alert). |
| Request ID | Unique identifier injected into API responses/logs for tracing. |
| ROI | Return on Investment percentage based on current value vs initial investment. |
| Fraction Owned | Portion of asset ownership represented by a token (<= remaining). |
| Sandbox Chain | Non-production blockchain environment for test mint operations. |
| Audit Log | Record of sensitive admin/system actions. |
| Utilization | Percentage of time assets are active/producing value. |
