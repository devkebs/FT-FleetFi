# FleetFi User Journey Map

FleetFi empowers three main user groups: Investors, Riders, and Operators.  
Each journey demonstrates how the platform turns tokenized EVs into inclusive income-generating assets.

---

## 1. Investor Journey
**Goal:** Earn ROI by co-owning EVs.
1. Sign up on the FleetFi dashboard.
2. Browse available tokenized EVs.
3. Buy fractional ownership shares (mock tokens).
4. Track revenue and swap activity from the dashboard.
5. Receive ROI updates (future blockchain integration).

---

## 2. Rider Journey
**Goal:** Earn daily income by driving an FT-managed EV.
1. Register with FleetFi fleet operator.
2. Get assigned a vehicle and access routes.
3. Swap batteries at biogas-powered swap stations.
4. Income auto-tracked and payout simulated.
5. Riders maintain vehicles; system updates status to investors.

---

## 3. Operator Journey
**Goal:** Manage fleet performance and uptime.
1. Access operator dashboard.
2. Monitor battery levels, charging stations, and status.
3. Adjust vehicle availability and manage swaps.
4. Approve maintenance and generate summary reports.

---

## User Experience Summary
Investor → FleetFi Platform → Fleet Operation → Swap Station → Rider → Revenue Distribution → Investor

---

**Key Differentiators:**
- Fractional EV ownership for inclusive investing.
- Biogas-powered swap stations for clean energy.
- Transparent digital tracking for trust and impact.
